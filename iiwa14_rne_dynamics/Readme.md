## iiwa14 RNEA 动力学控制概要

### 背景：
控制机械臂点到点到达指定位置抓取目标物体

### 期望末端位姿
- 位置（m）：`[0.62, -0.103, 0.796]`
- 姿态（四元数 w, x, y, z）：`[0.597, -0.115, 0.748, -0.267]`
### 实现流程（多问AI，AI能解决大部分问题）
1. 逆运动学求解目标关节角度。
一个供参考的教程：（https://colab.research.google.com/github/google-deepmind/mujoco/blob/main/python/least_squares.ipynb）
2. 关节空间五次多项式规划，确保初末速度和加速度为 0，实现平滑轨迹。
3. 力矩控制执行：RNEA 前馈（参照Matlab代码） + PD 反馈，并记录两部分的数值。
4. 绘制期望与实际关节运动曲线，检查跟踪效果。
5. 绘制 RNEA 前馈项和 PD 反馈项曲线，分析控制贡献。
